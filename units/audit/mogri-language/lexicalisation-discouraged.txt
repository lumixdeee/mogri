LEXICALISATION-DISCOURAGED.txt
==============================

Purpose
-------
This file lists languages for which introducing or forcing a lexical
mogri-equivalent is discouraged.

This is not a claim of deficiency.
It records cases where lexicalising mogri would reduce clarity,
damage existing expressive systems, or violate native usage norms.

KEY
---
D1 = Context-first language
     Meaning primarily carried by context, particles, omission,
     or discourse structure rather than lexical primitives.

D2 = Aspect / morphology already does the work
     Persistence-through-change is handled by aspectual systems,
     verb derivation, or suffix chains; a new carrier would interfere
     or duplicate.

D3 = Expressive or cultural resistance
     Language prefers narrative, poetic, theological, or relational
     framing; abstraction via a neutral carrier would flatten meaning.

NOTES
-----
- Multiple reasons may apply; only the dominant one is listed.
- “Discouraged” does not mean impossible, only unjustified.
- English is intentionally excluded:
  English is classified separately as “mogri latent”.

Status
------
This list is considered stable unless a native-speaker counterexample
is demonstrated and agreed.
